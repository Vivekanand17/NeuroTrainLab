# NeuroTrain Lab — frontend (Vite + React)

```bash
npm install
npm run dev      # dev server http://localhost:3000
npm run build    # production output → dist/
npm run preview  # preview production build locally
```

**API URL:** set `VITE_API_URL` in `.env` or `.env.local` (see `.env.example`). On Render, add `VITE_API_URL` as an environment variable before building the static site.
